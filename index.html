Estructura básica de un documento html

Elementos HTML


Un elemento html está formado por:

una etiqueta de apertura + un contenido + una etiqueta de cierre


<nombre de la etiqueta > Aquí va el contenido </nombre de la etiqueta >

< !DOCTYPE html >

Todos los documentos HTML deben comenzar con una declaración. La declaración no es una etiqueta HTML. Es una "información" para el navegador sobre el tipo de documento.

< html >



< /html >

Define un documento html. Todo lo que se encuentra dentro de estas etiquetas constituye nuestra página web es decir nuestro documento html.

< head >



< /head >

Contiene metadatos, dentro de este elemento se anidan elementos que aportan
información sobre el documento.

< body >



< /body >

Define el cuerpo del documento, es la parte visible en el navegador.

< title >



< /title >

Define el título de la página web. Se mostrará en la pestaña superior del navegador.

< p >



< /p >

Define un párrafo. El texto entre la etiqueta de apertura y la etiqueta de cierre es el contenido que será visible en el navegador.
